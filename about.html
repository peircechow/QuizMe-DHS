About QuizMe@DHS

QuizMe@DHS is a webapp developed for Dunmanians to gain more knowledge about the school.

How it works? [Gameplay instructions]

At the home page, questions will be loaded randomly in no particular order.Users can climb the leaderboard by either answering questions or submitting questions. Correct answers will be awarded more than incorrect answers. Submitiing questions will allow users to get a significant number of points. However, marks will be deducted if the user did not select an option or if the user submits an invalid question.
For more details, please refer to the point system below.

Point system

Action Points awarded
Correct answer 4
Incorrect answer 1
Did not select option -3
Submit a question 15
Submit Invalid Question -5
